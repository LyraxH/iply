# iply
 simple ip grabber, info locator, and security tools

## usage
   python iply.py

#### (g)et
   Prints current ip
#### (f)ind
   Finds and returns basic IP information on any valid IP. Or (m) to print details on your own ip
#### (s)ecurity
   Uses VirusTotal to find security information on any valid IP.
#### (v)erbose
   Prints all information available on any valid IP. (Basically spam to make the program look more useful, will probably be phased out later)
#### (q)uit 
   Exits the program
